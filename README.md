## Usage

Clone the repo into an empty project and run the following commands

```
npm init -y
npm start
```

List of scripts:

```
{
    "test" : "nodemon index.js",
    "start" : "node index.js"
}
```

Install nodemon locally using `npm install --save-dev nodemon`. Add the `-g` flag to install globally
